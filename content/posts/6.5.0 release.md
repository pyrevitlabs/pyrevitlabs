## pyRevit v6.5.0

Thanking all the great contributors, reviewers, actors, action figures of the past months in no particular order: Antoine Louchet, BIM4GIB, Biminent, Biseuv Olzhas, Chris Crosley, Da Bimci, DamirV, dosymep, Fred, gokul-pascalls, Jean-Marc Couffin, npt-1707, Roi Hason, Roman Golev, Andrea Ghensi, Swichllc, tay.0, Tee_Permpong, thumDer, Wurschdhaud

### Highlights

* C# loader significantly matured: faster startup, improved caching, and richer instrumentation
* Major CI/CD refactor: modular pipelines, signed releases, and cleaner artifact flow
* New dockable UI: Custom Properties pane
* Korean localization completed across UI and 286 bundle files
* New extensions added: BMT, Seed43, P13, Wood Framing
* Revit 2027 compatibility updates

---

### Performance & C# Loader

* Startup optimizations via attachment caching and reduced rebuild triggers
* Assembly cache now keyed on semantic version only (prevents unnecessary invalidations)
* UI build timing instrumentation added for profiling loader stages
* SmartButton initialization unified for reduced overhead
* Improved fallback handling for legacy Revit versions (2019/2020)

---

### Logging & Diagnostics

* Persistent loader logs via ScriptLoggerService (no more transient NLog loss)
* Output window behavior stabilized (no forced opening during startup)
* Fixed stylesheet resolution + default fallback handling
* DevTools log viewer reliability fixes
* Emoji/encoding handling corrected across logger paths

---

### CI/CD & Release Pipeline

* Full migration to modular C# build pipelines
* Prebuilt binaries now distributed via CI artifacts (bin/ removed from repo)
* Trusted signing restricted to release paths only
* NuGet cleanup and artifact pruning added
* Improved release validation and issue notification flow

---

### UI & Tools

* New Custom Properties dockable pane
* Keynote Manager improvements: wrapping, sequencing stability, expand/collapse behavior
* Sheet renaming tools upgraded with batch modes and locale-aware PDF renamer
* Extension manager enhancements and optional reload support
* Ribbon icon sizing and inheritance made configurable
* Section Box Navigator crash resolved

---

### Extensions

* Added: BMT, Seed43, P13, Wood Framing
* Extension registry updates and improved loading flexibility

---

### Revit API / Core Library

* Expanded parameter and category query helpers
* Improved linked model selection and pick tools
* Cloud/BIM 360 path support extended (Revit 2024+)
* External event handling fixes and context restoration improvements
* Filter and selection performance improvements

---

### Localization

* Full Korean UI localization added
* 286 bundle files translated (ko)
* YAML formatting and multi-line handling fixes
* Locale-aware improvements in PDF renaming tools

---

### CLI & Configuration

* Better Revit product resolution in CLI
* INI parsing centralized and hardened
* Telemetry escaping issues fixed

---

### Bug Fixes & Polish

* Dialog cancellation handling improved
* WPF output pane title fix
* Chart initialization race condition resolved
* Clipboard and parameter definition edge cases fixed
* Duplicate code cleanup and deprecated scripts removed
* Repo-shipped Revit API DLLs removed (~506 MB reduction)

---

### ⚠️ Deprecation Notice

Inline script metadata parsing is now deprecated.
A safer and more explicit configuration approach replaces it, and future versions will progressively remove inline-based metadata handling.

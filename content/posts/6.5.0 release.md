## pyRevit v6.5.0 (from v6.4.0 → v6.5.0)

* C# loader significantly matured: faster startup, improved caching, and richer instrumentation (#3388, #3397, #3420, #3439)

### ⚠️ Deprecation Notice

Inline script metadata parsing is being phased out (#3397).

While still supported, it is now disabled by default in the C# loader to improve startup performance. Extension authors are encouraged to move metadata into standard bundle configuration files, as future releases will continue the transition away from inline metadata parsing. Helpers are available in the pyRevitDev Extension.
